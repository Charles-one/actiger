2019.11.30

1. Kubernetes扩容增加node节点

创建token


kubeadm token create

查看kubernetes认证的SHA256加密字符串是多少


openssl x509 -pubkey -in /etc/kubernetes/pki/ca.crt | openssl rsa -pubin -outform der 2>/dev/null | openssl dgst -sha256 -hex | sed 's/^.* //'

加入master节点


kubeadm join <主master ip>:6443 --token <刚刚创建的token>     --discovery-token-ca-cert-hash sha256:<上面的加密字符串>

操作这些时先更新下系统.
