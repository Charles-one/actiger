1. 要求: 编写一个输入复制到输出的程序, 将其中多个空格用一个空格代替.

2. 分析.

1. 比较前一个字符和当前字符如果同为空(’ ‘), 则不打印, 否则打印.

3. 代码.

#include<stdio.h>
int main(){
    int a, alast;
    alast=0;
    while((a=getchar()) !=EOF){
        if(a==' ' && alast ==' '){
            continue;
        }
        else{
            putchar(a);
        }
        alast = a;
    }
}
