1. 要求: 编写一个程序打印摄氏温度转换为相应华氏温度的转换表.

2. 分析.

1. 由C = (5⁄9)(F-32)得到F = C(9⁄5)+32

2. C摄氏和F华氏温度都有小数, 可使用float

3. 代码

注意事项: %6.1f表示6个数字宽右对齐且保留1位小数

#include<stdio.h>
int main(){
    float f;
    float c;
    int step = 20;
    int minc = -20;
    int maxc = 300;
    c = minc;
    printf("摄氏温度转换华氏度\n");
    while( c <= maxc ){
        f = c * 9.0 / 5.0 + 32;
        printf("%3.0f %6.1f\n", c, f);
        c = c + step;
    }
    return 0;
}
